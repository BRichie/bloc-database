3. Describe one difference between SQL and other programming languages.

ANSWER: SQL is a declarative language unlike most other programming languages which are procedural.  With procedural algorithms must be defined In order to search the database, with SQL the programmer defines his search criteria and the engine determines the most efficient algorithm to return the desired data.
